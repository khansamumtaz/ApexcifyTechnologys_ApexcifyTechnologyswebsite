# ApexcifyTechnologyswebsite
A modern, responsive, and premium-looking company website built using **HTML, CSS, and JavaScript**.  
This project was created as part of an internship/web development task to showcase professional front-end design, interactivity, and responsiveness.

##  Project Overview
**ApexcifyTechnologys Website** is a professional business-style landing website designed to represent a modern technology company.  
It includes visually polished sections, smooth animations, responsive layouts, interactive gallery features, and a clean user experience.
The goal of this project is to create a **company-level premium website UI** using only **frontend technologies**.

##  How to Run the Project
### Method 1: Run Locally
1. Download or clone the repository
2. Open the project folder
3. Double-click on `index.html`
OR
### Method 2: Open with Live Server
1. Open the project in **VS Code**
2. Install the **Live Server** extension
3. Right-click on `index.html`
4. Click **"Open with Live Server"**
